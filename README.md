# LandingPage

Static landing pages and product demos.

## Sites

- `sites/saboriza-opc` - Portuguese Saboriza OPC landing page.
- `sites/studio-ia-product-demo-cn` - Chinese product demo prototype.

## Vercel

Create one Vercel project per landing page and set the project root directory:

- `sites/saboriza-opc`
- `sites/studio-ia-product-demo-cn`

Use framework preset `Other`, with no build command and output directory `.`.
